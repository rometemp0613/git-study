## 오늘 배운 것

### 자동 테스트 (PR마다 테스트 실행, Status Check)

1. **`on: pull_request`** 트리거로 PR 시 자동 테스트 실행
2. **Status Check** — 워크플로우 결과가 PR 페이지에 표시됨
3. **Required Status Check** — Ruleset에서 설정, 테스트 실패 시 머지 차단
4. **exit code** — exit 0=성공, exit 1=실패 (hooks와 동일 원리)
