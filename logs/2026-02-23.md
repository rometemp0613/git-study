## 복습
- upstream 동기화 순서: `git remote add upstream` → `git fetch upstream` → `git merge upstream/main` → `git push`
- `git pull` vs `git fetch`: pull은 merge까지, fetch는 원격 브랜치만 업데이트
- PR 머지 3가지: Merge commit / Squash and merge / Rebase and merge
