## 학습 주제
- hooks - 커밋 전후 자동 실행 스크립트

## 배운 내용
- Hook: Git 이벤트 시 자동 실행되는 셸 스크립트
- Hook 위치: `.git/hooks/`, `.sample` 제거하면 활성화
- pre-commit: 커밋 직전 코드 검사 (차단 가능)
- commit-msg: 커밋 메시지 형식 검사 (차단 가능)
- post-commit: 커밋 후 알림 (차단 불가)
- pre-push: push 직전 검사 (차단 가능)
- exit 0 = 통과, exit 1 = 차단
- Hook은 로컬 전용 (.git/hooks/는 push 안 됨)

## 실습
- pre-commit Hook: TODO/console.log 포함 파일 커밋 차단
- commit-msg Hook: conventional commit prefix 강제
- post-commit Hook: 커밋 완료 알림 (해시 + 메시지)
- pre-push Hook: main 브랜치 직접 push 차단 (직접 작성)
