# Git 학습 프로젝트

이 저장소는 Git & GitHub 학습용입니다.

## 대화 시작 시 할 일

1. README.md를 읽어서 현재 진도를 확인한다
2. 체크리스트에서 아직 완료하지 않은 항목 중 다음으로 배울 주제를 권유한다
3. 사용자가 학습을 시작하면 해당 주제를 친절하게 설명한다

## 학습 완료 후 할 일

1. 배운 내용을 해당 단계의 notes 폴더에 정리한다
2. logs 폴더에 날짜별 학습 일지를 작성한다
3. README.md 체크리스트를 업데이트한다
4. 변경사항을 커밋하고 push 여부를 물어본다

## 파일 구조

- `README.md` - 진도 체크리스트 및 학습 일지
- `logs/` - 날짜별 학습 일지
- `notes/` - 단계별 학습 노트
